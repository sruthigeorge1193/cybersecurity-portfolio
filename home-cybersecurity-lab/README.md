# Home Cybersecurity Lab Setup

## Overview
To support my cybersecurity learning and portfolio projects, I built a small home lab using VirtualBox. The goal was to create an isolated environment where I could safely generate security events, analyse logs, inspect network traffic, and experiment with security tools without affecting my primary system.

This lab environment is the foundation for several projects in this portfolio, including Windows log investigation and network traffic analysis.

---

## Lab Environment
- Virtualization: VirtualBox
- Guest OS: Windows 10
- Network Mode: NAT (isolated but internet-enabled)

### VM Configuration
- RAM: 4 GB
- CPU: 2 cores
- Storage: 50 GB

---

## Tools Installed
- Windows Event Viewer (log investigation)
- PowerShell (system inspection)
- Wireshark (network traffic analysis)

---

## What I Learned
Building this lab helped me understand:
- How virtualization supports safe security testing
- How to configure and manage isolated environments
- How to set up systems for log and network investigations
